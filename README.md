# bratlien_uibank

Public working repository for the UiBank hackathon project, including automation work, site research, and Codex-assisted implementation notes.

## Goals

- Build and refine a UiBank registration workflow.
- Capture site context, field mappings, and behavior notes so automation logic is grounded in the actual UI.
- Keep project planning, research, and implementation artifacts in one place.

## Repository Structure

- `docs/` project plans, checklists, and implementation notes
- `research/` site analysis, field maps, and workflow observations
- `references/` supporting reference material for prompts, agent notes, or future skill-related docs
- `workflows/` UiPath workflow projects and automation assets

## Getting Started

1. Use `docs/project-checklist.md` as the working task list.
2. Capture UiBank page and field observations in `research/`.
3. Build the registration automation under `workflows/`.
4. Use short feature branches and PRs for meaningful changes.

## Current Focus

- Establish the project structure
- Define the UiBank registration workflow scope
